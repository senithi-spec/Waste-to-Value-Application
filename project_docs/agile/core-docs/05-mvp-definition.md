# 05 - MVP Definition

The MVP focuses on enabling core waste exchanges with simple, reliable workflows.

Included Features (MVP):
- User registration & login (generator, collector, recycler)
- Create and browse waste listings (type, qty, unit, location, desc)
- Request and approve/reject workflow
- Real-time notifications and listing updates via Socket.io
- Basic admin functions and seeded test users
- Dockerized backend and frontend with migration/seed scripts

Out of scope for MVP:
- Payment gateway, advanced analytics, enterprise-grade scaling, mobile apps.
